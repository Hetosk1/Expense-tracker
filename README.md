## Installation Guide
1. First of all I spend more than 3 weeks coding this thing so please star this repository so that I can impress my crush üòÅ 

2. Clone the repository by the following command 
  ```
  git clone https://github.com/Hetosk1/expense-tracker
  ```

3. Navigate into directory & Install the depencies by the command 
  ```
  cd expense-tracker
  npm install
  ```

4. Start the Application
  ```
  npm start
  ```
